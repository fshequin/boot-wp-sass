== Description ==

Adds a page under Tools menu that shows APC stats, opcode entries, user cache key/values etc.. WordPress compatible version of apc.php from [pecl apc package][1].

More info available at [here][2].

[1]: http://pecl.php.net/package/APC
[2]: http://whitewp.com/wp-apc-panel-plugin-manage-apc-cache-in-wordpress-admin

== Installation ==

Upload WP APC Panel to your wordpress site, or install it via New Plugin page then activate it.
